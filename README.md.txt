# 🛡️ Visitor Monitoring System

A security-focused system that captures visitor data using Raspberry Pi, Pi Camera, and Python to ensure safe entry logging and automated gate response.

## 🔧 Technologies Used
- Raspberry Pi
- Python
- OpenCV (for image capture)
- GPIO
- DC Motor (for gate mechanism)

## 📸 Features
- Captures visitor images with time/date stamp
- Automatically controls gate using a DC motor
- Stores visit logs locally

## 🎯 Objective
Designed to improve home/office security by tracking and recording visitor entry. Demonstrates real-world integration of hardware and software.
